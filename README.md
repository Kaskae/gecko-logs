
# Gecko Logs 🦎

Ferramenta simples em Bash para análise e filtragem de logs,
voltada para estudos de segurança da informação e forense básica.

Pretendo atualizar essa ferramenta mais para frente, trazendo uma versão mais nova dela para python.
Além de possibilitar que o usuário faça buscas por Logs mais complexos e diferentes dos que foram apresentados.


## Funcionalidades
- Contagem de IPs
- Filtro de ocorrências
- Análise básica de logs
